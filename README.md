互联网使得我们可以轻松的从网上获取到各种资料，但也产生了新的问题，如何能找到优质的资料呢？
尤其是合适的教程。

这个项目致力与寻找并分享优质的共享资料。
共享并不是说免费，但其至少要保证可以在没有付钱或其他特殊条件的前提下可以正常使用。

### [Build Your Own Lisp](http://buildyourownlisp.com/)[EN,CN]

有[中文翻译](https://ksco.gitbooks.io/build-your-own-lisp/)，但不完整，好像已经废弃了。
